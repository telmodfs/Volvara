POLÍTICA DE
PRIVACIDADE

Última actualização: Março 2026  ·  Volvara · NIF 227192753 · Portimão, Portugal

Resumo simples: recolhemos apenas o que precisamos para te responder. Não vendemos dados a ninguém. Podes pedir a eliminação dos teus dados a qualquer momento.

1. Quem somos

Volvara é uma web agency com sede em Portimão, Algarve, Portugal. NIF: 227192753. Email de contacto: go@volvara.pt.

Somos o responsável pelo tratamento dos dados pessoais recolhidos através deste website, nos termos do Regulamento Geral sobre a Proteção de Dados (RGPD — Regulamento UE 2016/679).

2. Que dados recolhemos

Através do formulário de contacto recolhemos:

  • Nome
  • Nome do negócio
  • Endereço de email
  • Tipo de negócio (seleccionado pelo utilizador)
  • Mensagem livre introduzida pelo utilizador

Não recolhemos dados de pagamento, dados sensíveis nem dados de menores.

3. Para que usamos os dados

Os dados recolhidos são utilizados exclusivamente para:

  • Responder ao pedido de contacto ou orçamento
  • Enviar a demonstração do website solicitado
  • Comunicações directamente relacionadas com o serviço pedido

Não utilizamos os teus dados para marketing não solicitado nem para fins comerciais além dos indicados acima.

4. Base legal para o tratamento

O tratamento dos dados baseia-se no consentimento expresso que nos concedes ao submeter o formulário (art. 6.º, n.º 1, al. a) do RGPD).

5. Quanto tempo guardamos os dados

Os dados de contacto são conservados pelo período necessário para concluir o processo comercial iniciado, ou até 12 meses após o último contacto, salvo obrigação legal em contrário.

Se nos pedires a eliminação dos teus dados, removemo-los no prazo de 30 dias.

6. Partilha com terceiros

Os teus dados são processados através do serviço Formspree (formspree.io), utilizado para recepção dos formulários, sujeito à sua própria política de privacidade compatível com o RGPD.

Não vendemos, cedemos nem partilhamos os teus dados com terceiros para fins publicitários ou comerciais.

7. Os teus direitos

Ao abrigo do RGPD, tens direito a:

  • Acesso — saber que dados temos sobre ti
  • Rectificação — corrigir dados incorrectos
  • Eliminação — pedir a remoção dos teus dados ("direito ao esquecimento")
  • Portabilidade — receber os teus dados em formato estruturado
  • Oposição — opor-te ao tratamento em determinadas circunstâncias
  • Retirada do consentimento — a qualquer momento, sem efeitos retroactivos

Para exercer qualquer destes direitos, contacta-nos por email: go@volvara.pt. Respondemos no prazo de 30 dias.

Tens também o direito de apresentar reclamação à autoridade de controlo portuguesa: CNPD (Comissão Nacional de Protecção de Dados) — cnpd.pt.

8. Cookies

Este website utiliza apenas cookies técnicos estritamente necessários para o funcionamento do site. Não utilizamos cookies de rastreamento, analytics ou publicidade de terceiros.

Se no futuro implementarmos ferramentas de analytics, esta política será actualizada e solicitaremos o teu consentimento.

9. Segurança

Implementamos medidas técnicas e organizativas adequadas para proteger os teus dados contra acesso não autorizado, perda ou divulgação indevida.

10. Alterações a esta política

Podemos actualizar esta política periodicamente. A data de última actualização está indicada no topo deste documento. Alterações significativas serão comunicadas de forma clara.
